About the Server

Here on Drakonnas we strive to deliver the best gaming experience.

Our team of highly trained IT professionals are always updating and extending the server and its features!

We provide an mostly bugfree and fully featured pvp gameplay with a nice team that is happy to help with any question.

You can join the server with mc.drakonnaspvp.com - so why don't you drop by for some hardcore pvp action?

Points System - earn points ingame to buy premium things like items or ranks

PvP Events - Take part in awesome monthly PvP events for points, diamonds, free ranks

Free rank upgrades

Donation ranks

Seasonal Map Changes - We decorate our server for seasonal events!
